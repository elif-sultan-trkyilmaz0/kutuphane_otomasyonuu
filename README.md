#  Kütüphane Otomasyon Sistemi

## Proje Hakkında

Bu proje, C++ ve Qt Framework kullanılarak geliştirilmiş bir kütüphane otomasyon sistemidir. Uygulama, kitap kayıtlarının, üye bilgilerinin ve ödünç alma/teslim işlemlerinin kolay bir şekilde yönetilmesini amaçlamaktadır. Veriler SQLite veritabanında saklanmaktadır.

## Özellikler

### Kitap İşlemleri

* Kitap ekleme
* Kitap güncelleme
* Kitap silme
* Kitap listeleme

###  Üye İşlemleri

* Üye ekleme
* Üye güncelleme
* Üye silme
* Üye listeleme

###  Ödünç İşlemleri

* Kitap ödünç verme
* Kitap teslim alma
* Ödünç kayıtlarını takip etme

## Kullanılan Teknolojiler

* C++
* Qt Framework
* Qt Designer (.ui)
* SQLite Veritabanı
* QSqlDatabase ve QSqlQuery

## Proje Dosya Yapısı

* `ana_pencere` : Ana menü ekranı
* `kitap_islemleri` : Kitap yönetim modülü
* `uye_islemleri` : Üye yönetim modülü
* `odunc_alma_islemleri` : Kitap ödünç verme işlemleri
* `odunc_teslim_islemleri` : Kitap teslim işlemleri
